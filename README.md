# lodash-prototype

Use lodash functions directly on the objects.

## Usage

In node:

```bash
require('lodash-prototype');
"10".parseInt(15); // should give 15
```

In browser:

```html
<script src="lodash.js"></script>
<script src="lodash-prototype.js"></script>
<script>
  "10".parseInt(15); // should give 15
</script>
```
